== Description ==

The Svenska Namnsdagar plugin for WordPress every day displays the Swedish names day on this day.
This plugin is provided by <a href="http://dejavuproduction.se">DejaVuProduction</a> and uses the site to fetch data.

You may install this plugin on your website.


== Installation ==

 Upload The Svenska Namnsdagar Plugin to your website and activate.

    - Add the widget to your sidebar or any widget area or use the shortcode [svenska_namnsdagar] on your post or page.

    - You're done.


== Frequently Asked Questions ==
= Why was the widget stuck on a certain day on how do I fix that?   =

If the svenska namnsdagar widget got stuck on a certain date you can try to deactivate the plugin and activate it again.
